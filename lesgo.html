Lesgo

Never waste time waiting for the bus again

The skinny

Overview

Duration / 3 weeks

Tools / Figma

Roles / UX Research, Content Strategy, Information Architecture, Visual Design

Deliveries / Research Plan, Competitive Analysis, User Survey, User Interviews, Personas, User Journey, User Stories, User Flow, Sitemap, Wireframes, Branding, Usability Testing, High-Fidelity Prototype

The brief

Problem

A local transportation agency has recently expanded and added numerous bus routes that happen to stop at a particular bus stop, which has led to commuter frustration and confusion.

The MVP

Outcome

A mobile application that satisfies client requirements and is tested to ensure that users know the what, when and how regarding bus line arrival and departure times at a particular bus stop.
